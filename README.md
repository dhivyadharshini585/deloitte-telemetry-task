# Deloitte Telemetry Format Conversion

This repository contains my solution for Task 1 of the Deloitte Australia Technology Virtual Experience Program (Forage).

## Objective

Convert telemetry data from two different JSON formats into a single unified format.

## Technologies Used

* Python 3
* JSON
* Unit Testing

## Features

* Converts Format 1 telemetry data to the target format.
* Converts Format 2 telemetry data to the target format.
* Handles timestamp conversion to milliseconds since epoch.
* Passes all provided automated test cases.

## Result

All unit tests passed successfully.
